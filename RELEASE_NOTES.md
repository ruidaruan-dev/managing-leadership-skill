# 📋 版本发布说明

## v2.0 - 关系网络管理系统 (2026-04-09)

### 🎉 重大更新

本次更新为管领导系统引入了全新的**关系网络管理模块**，从单一的领导画像工具升级为完整的领导关系管理系统。

### ✨ 新增功能

#### 1. 多主体关联建模
- 🗂️ **领导档案管理**：支持创建和管理多个领导档案
- 🧠 **精细化画像**：性格特征、行为模式、偏好喜好、规避禁忌
- 🎭 **16种原型匹配**：自动关联已有的领导原型数据库
- 📊 **关系状态跟踪**：亲密盟友/正常同事/谨慎相处/关系紧张

#### 2. 交互式任务看板
- 📋 **任务管理**：创建、完成、删除互动任务
- ⚡ **智能排序**：按优先级（紧急/重要/普通）+ 时间双重排序
- 🔴 **逾期预警**：超时任务红色高亮提醒
- 📈 **统计面板**：关系人数/待办/逾期/已完成一目了然

#### 3. 预设档案快速创建
- 🚀 **6种预设模板**：
  - 👔 直属领导
  - 🏢 部门负责人
  - 🤝 跨部门协作者
  - 💻 技术型领导
  - 💰 项目赞助人
  - 🔀 虚线汇报领导
- 📝 **智能填充**：自动填写职位、关系状态、行为模式等字段
- ✍️ **灵活编辑**：所有预设字段均可继续修改

#### 4. 智能策略推荐
- 🎯 **6种互动目标**：对齐目标/汇报进展/申请资源/关系建设/修复关系/谈判协商
- 💡 **自动建议**：根据互动目标生成针对性策略要点
- 📚 **原型策略**：基于领导原型的AI推荐互动策略

#### 5. 全局关系网络视图
- 📊 **任务看板**：全局查看所有领导的待办任务
- 👥 **领导列表**：卡片式展示，快速访问
- 🔗 **快速跳转**：点击领导名称直接进入详情

### 📖 使用指南

1. **访问系统**：启动开发服务器后，点击「🗂️ 关系网络」标签
2. **创建档案**：点击「+ 建立新档案」，选择预设模板或手动填写
3. **管理任务**：进入领导详情页，添加互动任务
4. **全局查看**：在「任务看板」页面管理所有待办
