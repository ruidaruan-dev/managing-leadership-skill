# 管领导 — Managing Leadership Skill

> **我们程序员受够了被各种东西夺取工作。我将把我失去的东西夺回来。**
>
> 管领导 Skill 补齐我们最后一块能力拼图——帮助我们这些不太会向上沟通的程序员争取资源，升职加薪！

在 AI 不断夺走人类工作的时代，聪明的程序员不是被 AI 替代的那个，而是**用 AI 来"管理"自己领导的那个**。

## 内置功能

### 16 种中国企业领导原型画像

覆盖国企（官僚型处长、技术转行政型、政治型、混日子型）、民企（创始人老板、职业经理人、家族接班人、草根创业者）、外企（海归VP、外籍高管、本土化高管、矩阵型领导）、互联网大厂（技术VP/CTO、产品型领导、空降型高管、内卷推动者）四大生态系统。每个画像包含核心动机、沟通风格、面子敏感度、AI态度、弱点盲区和管理策略。

### 西方管理理论 × 中国关系学融合框架

将 Drucker 管理上级理论、Goleman 领导力风格、情境领导模型与中国关系动力学（关系/面子/人情/圈子/中庸）深度融合。让你用西方理论看懂局面，用中国关系学落地执行。

### 10 大高频职场困境场景模板

每个场景配备多种风险等级的策略选项和可直接使用的中文参考话术：
- 领导让你做非技术工作
- **领导要用 AI 替代团队成员**（核心场景）
- 汇报项目延期
- 争取资源/HC
- 薪资/晋升谈判
- 领导抢功
- 多线领导之间周旋
- 新领导到任建立关系
- 绩效评审准备
- 被要求评价同事

### 程序员专属职场生存手册

不是泛泛的职场鸡汤，而是专门为**技术人员量身定制**的实战指南：
- 技术能力如何转化为政治货币
- 工期博弈的锚定-缓冲-谈判模型
- 技术债务如何变成争取资源的杠杆
- AI 威胁的三种战略定位（AI 冠军 / 理性顾问 / 静默对冲）
- IC vs 管理路径抉择
- 1-on-1 策略手册
- 薪资/晋升/退出策略

### 沟通起草引擎

选择领导类型 + 沟通场景 → 自动生成针对该领导原型优化的邮件/消息文稿，附带策略注解解释每段话的政治目的。8 种场景 × 16 种领导 = 128 种定制组合。

### 可视化操作前端

内置 React + Tailwind 暗色主题操作界面，四大交互模块：
- **关系网络管理** 🆕：多领导档案精细化管理 + 交互式任务看板
- **领导画像生成器**：输入领导特征 → 匹配原型 → 查看完整档案和策略
- **场景分析器**：选择困境 → 获取多策略选项 + 参考话术
- **沟通起草助手**：选领导 + 选场景 → 一键生成定制文稿

## 快速开始

### 作为 Qoder/Agent Skill 使用

将本仓库克隆到你的 skills 目录：

```bash
git clone https://github.com/ruidaruan-dev/managing-leadership-skill.git ~/.qoder/skills/managing-leadership
```

然后在 Qoder 中直接描述你的职场困境，Skill 会自动激活。

### 启动操作前端

```bash
cd web
npm install
npm run dev
```

访问 `http://localhost:5173` 即可使用。

#### 关系网络管理系统使用指南

1. 点击顶部导航栏「🗂️ 关系网络」标签
2. 在「我的领导」页面点击「+ 建立新档案」
3. 选择预设模板快速创建，或手动填写领导信息
4. 进入领导详情页，添加互动任务和管理策略
5. 在「任务看板」页面全局查看所有待办任务

### 运行画像匹配引擎（Python）

```bash
python scripts/leader_profiler.py
```

## 核心理念

**这不是一个教你拍马屁的工具。**

这是一个帮助技术人员理解职场权力动力学的系统化框架。它的底层逻辑是：

1. **识别**领导原型（用分类体系）
2. **分析**他们的动机（用西方管理理论）
3. **执行**你的策略（用中国关系学）
4. **放大**你的优势（用 AI 增强）

程序员最大的职场悲剧不是技术不行，而是明明技术很行，却因为不会"管理"领导而错失资源、错过晋升、最终被用 AI 替代。

**管领导 Skill，让你把失去的东西夺回来。**

## 伦理声明

本工具的所有策略建议严格限于合法合规的职场博弈范围内。我们鼓励的是理性的向上沟通和策略性的关系管理，而非操纵、欺骗或任何形式的不道德行为。
